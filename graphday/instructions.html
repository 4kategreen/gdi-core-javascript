Project Day: Graphs

Array of Objects

Make an array of objects that you can graph later on. This can be anything that will look nice on a graph. Ages, height, baseball team championships. Be creative! 

        
          var myObjects = [
            {
              name: 'Kate',
              type: 'human'
            }, {
              name: 'Sara',
              type: 'dog'
            }, {
              name: 'Pumpkin',
              type: 'cat'
            }
          ];
Find my example of this exercise here!
  1. Create an html file where you can open it in your browser.
  2. Create a script tag at the bottom of the body (before it closes). You'll work here
  3. Create an array of objects that you thought up earlier.
  4. Using the for loop, iterate over your objects and create a string with your object's data that you'll append to the html
  5. Create an element, put the string you created in the new element's html and then append it to the document.body
        
          var myArray = ['this','that','something'],
              myString = '';

          for (var i=0; i<myArray.length; i++) {
            myString+= 'my array value is ' + myArray[i];
          }

          var ele = document.createElement('div');
          ele.html = 'This value is ' + myString + '. ';
          document.body.append(ele);

Graphs!

Next, we'll make a graph of the data we just created. This requires a third party library, ChartJS. Check out the site, see if you can figure out what you'll need to do to make a chart of your own. Check out the documentation for the type of graph you want to make, see how the data needs to be organized.

Check out my example. View the source and check out my function exercise2(). This will give you an idea of what you'll be doing (though I did a bit more than you need to).

  1. First, add a script tag above the one you made for the first exercise. Make the src: http://kategreen.us/gdi/graphday/js/Chart.js. Don't forget to close the tag!
  2. Next, add a canvas tag with an id in your html file.
  3. Experiment with creating a chart with the example data in the documentation. Once you make it work in your file, you can move on to using your own data.
    4. 
          var ctx = document.getElementById('myGraphCanvasChangeMe').getContext("2d"),
              myLineChart = new Chart(ctx).Line(data, options);

    You don't need to have the options variable here unless you're doing something with it!

  4. We need to make our data consumable by the chart app, so we will need to make new objects. You can do it by hand or manipulate the one you made before, like I did. The second one is more of a challenge. Figure out what you need to do to put your data into the data variable.
